electron-windows-notifications Sample

This sample demonstrates the use of the electron-windows-notifications module. Use the button below to create various different notifications. Demo is based on this great sample: https://github.com/hokein/electron-sample-apps/tree/master/notifications

Basic notification: title only.

Image notification: title, text & image.

Actions notification: text, image and action buttons.

Tile notification: update the app's tile.
